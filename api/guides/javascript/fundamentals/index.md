## Learning Path

**Recommended order for beginners:**

1. Start with [Variables](/guides/javascript/fundamentals/variables)
2. Learn [Strings](/guides/javascript/fundamentals/strings) and [Numbers](/guides/javascript/fundamentals/numbers)
3. Master [Arrays](/guides/javascript/fundamentals/arrays) and [Objects](/guides/javascript/fundamentals/objects)
4. Understand [Functions](/guides/javascript/fundamentals/functions) and [Scope](/guides/javascript/fundamentals/scope)
5. Practice with [Conditionals](/guides/javascript/fundamentals/conditionals) and [Loops](/guides/javascript/fundamentals/loops)
6. Level up with ES6+ features
