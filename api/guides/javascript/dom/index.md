# DOM Manipulation

The Document Object Model (DOM) is your gateway to creating interactive web pages. Learn how to select elements, modify content, and respond to user actions.

---

## Core Concepts

- [Selectors](/guides/javascript/dom/selectors) - Finding elements on the page
- [Manipulation](/guides/javascript/dom/manipulation) - Modifying content and attributes
- [Events](/guides/javascript/dom/events) - Responding to user interactions

---

## Learning Path

1. Start with [Selectors](/guides/javascript/dom/selectors) - learn to find elements
2. Learn [Manipulation](/guides/javascript/dom/manipulation) - change the page dynamically
3. Master [Events](/guides/javascript/dom/events) - make your pages interactive

---

Before learning DOM manipulation, understand:
